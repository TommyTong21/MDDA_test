# Routing Collab Demo

物流多约束路径规划与承运协同系统。本工程同时作为 MDDA 框架的官方端到端示例产品。

## 产品背景
- **核心痛点**：物流排线人工效率低、时窗/冷链等多约束冲突；异常上报滞后，承运沟通链路长。
- **业务目标**：实现订单整形 → 约束求解（或模拟求解） → 协同广播 → 异常回填 的闭环。
- **角色**：调度员（发起规划）、承运方（接单、反馈异常）。

## 目录结构
- `docs/`: 存放产品各阶段的沉淀文档（PRD、ADR 等）
- `src/`: 核心业务代码（Next.js App Router）
- `supabase/`: 数据库与 Edge Functions 配置
- `.trae/`: Trae IDE 相关的智能配置与资产
