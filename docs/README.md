# Routing Collab Demo - 项目文档

本目录用于存放 `routing-collab-demo`（物流多约束路径规划与承运协同系统）在 MDDA 工作流中产出的各阶段文档。

## 目录说明

- **prd/**: 产品需求文档（Phase 1 产物），包含竞品分析、用户画像及需求拆解。
- **adr/**: 架构决策记录（Phase 2 初始产物），记录技术选型、组件规范和物理约束。
- **api-contract/**: 前后端接口契约（Phase 2 协作产物），定义边界数据结构。
- **qa/**: 质量保证与测试报告（Phase 3 部署前产物）。
- **user/**: 用户使用手册及操作指南。

> 注意：所有文档由 MDDA 体系下的各个 Agent 按照 `.trae/blocks/` 中的模板生成，并在关键确认点（CP1-CP8）由人工 Review。
